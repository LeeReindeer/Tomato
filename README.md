# Tomato

> Tomato is a HTTP web framework written in Java 8.
>
> Thanks [Gin](https://github.com/gin-gonic/gin) for the great design.

- Java8 features used
- Function Style URL Mapping
- MVC
- IOC
- AOP
- Embeded Tomcat
